Administrer les utilisateurs (droits, rles...)

De mme, le module User est requis ds l'installation et fournit les fonctionnalits lies la gestion de compte et aux droits d'accs.

Chaque module fournit une liste de permissions disponibles.

Un rle est un ensemble de permissions.

Un utilisateur peut avoir plusieurs rles, ses droits d'accs seront donc dfinis par la somme des permissions de chacun de ses rles.

Des modules peuvent ajouter ou affiner les permissions d'autres modules.

Comprendre la licence Drupal

Drupal, un logiciel open-source : La totalit des sources de Drupal sont disponibles au public sous licence GNU GPL v2 (License gnrale publique). Cela veut dire qu'on :

  • est libre d'excuter le logiciel, pour n'importe quel usage
  • est libre d'tudier le fonctionnement d'un programme et de l'adapter ses besoins, ce qui passe par l'accs aux codes sources
  • est libre de redistribuer des copies
  • a l'obligation de faire bnficier la communaut des versions modifies

Les sources sont donc redistribues sur Drupal.org. Si on distribue une version modifie de Drupal, on a l'obligation de rendre son code source public. En revanche, ce n'est pas le cas pour une utilisation personnelle.

Installation de Drupal avec Drush

drush dl drupal  # tlcharge la dernire version de Drupal
mv drupal-7.* monsite  # renomme le dossier contenant Drupal
cd monsite  # se place dans ledit dossier

# Installe Drupal dans la base 'mabase'

drush en l10n_update  # Tlcharge et installe le module l10n update
drush l10n-update  # Vrifie et tlcharge les traductions

drush cc all  # Vide les caches
